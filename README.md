# PuyoPuyo

Cocos2d-x で作成したぷよぷよ。  
ぷよぷよは小学生頃にはまった大好きなゲームです☆  

![demo](https://github.com/okamura0510/PuyoPuyo/blob/master/demo.gif)

## Requirement

Cocos2d-x 3.3.0  
Windows7  
  
※WindowsとiPhone5のみ動作確認を行いました(それ以外は動作保障外) 

## Usage

* 左右キー  
左右移動

* 下キー  
下移動

* Xキー/上キー  
右回転

* Zキー  
左回転

* Retry!ボタン  
ゲームリトライ

* バーチャルキーボード  
実機の場合はこれを使用
